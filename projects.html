Projects

RiptFitness

Collaborated with a team to design and develop Ript, a full-stack fitness application that enables users to create and store personalized workouts, track calorie intake, and engage with a built-in social media feed. The backend was developed using Java Spring Boot, with MySQL for data management, and deployed on Azure for scalability and performance. The frontend, built with React Native, ensures a seamless cross-platform experience for both iOS and Android users.

My contributions focused on backend development, specifically implementing key features such as:

  • Workouts: Enabling users to create, update, and manage their fitness routines.
  • WorkoutData: Handling the storage and retrieval of workout performance data, including reps and weight, to generate insightful progress graphs.
  • Streak Features: Developing functionality to track user consistency and maintain motivation through streak tracking.

This project provided valuable experience in full-stack development, cloud deployment, and working collaboratively in an Agile development environment.

Movie Review API

Developed a solo project for a software engineering course, leveraging Python and FastAPI to create a web application that fetches movie reviews from YouTube using the Google YouTube API. The application allows users to retrieve full video descriptions by video ID and supports multiple languages, enhancing accessibility and user experience.

Key Features:

  • YouTube API Integration: Efficiently retrieves movie review videos and extracts comprehensive metadata.
  • Video Description Retrieval: Provides detailed video descriptions based on unique video IDs.
  • Multilingual Support: Enables the processing and presentation of reviews in various languages to accommodate a diverse user base.

This project demonstrates proficiency in developing RESTful web services, integrating third-party APIs, and implementing scalable backend solutions using modern frameworks.
