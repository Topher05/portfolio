Projects

RiptFitness

For this project, I collaborated with a team to design and develop Ript, a full-stack fitness application that empowers users to create and store personalized workouts, track calorie intake, and engage with an integrated social media feed. The backend was implemented using Java Spring Boot, with MySQL for efficient data management, and deployed on Azure, ensuring scalability and high performance. The frontend, built with React Native, delivers a seamless and responsive cross-platform experience for both iOS and Android users.

My contributions focused on backend development, specifically implementing key features such as:

  • Workouts: Enabling users to create, update, and manage their fitness routines.
  • WorkoutData: Handling the storage and retrieval of workout performance data, including reps and weight, to generate insightful progress graphs.
  • Streak Features: Developing functionality to track user consistency and maintain motivation through streak tracking.

This project enhanced my expertise in full-stack development, cloud-based deployments, and Agile development methodologies, while strengthening my skills in backend development and API design.

Movie Review API

I developed a solo project for a software engineering course, leveraging Python and FastAPI to create a web application that fetches movie reviews from YouTube using the Google YouTube API. The application allows users to retrieve full video descriptions by video ID and supports multiple languages, enhancing accessibility and user experience.

Key Features:

  • YouTube API Integration: Efficiently retrieves movie review videos and extracts comprehensive metadata.
  • Video Description Retrieval: Provides detailed video descriptions based on unique video IDs.
  • Multilingual Support: Enables the processing and presentation of reviews in various languages to accommodate a diverse user base.

This project demonstrates proficiency in developing RESTful web services, integrating third-party APIs, and implementing scalable backend solutions using modern frameworks.

Lemniscate

I created a fast-paced, top-down 2D game, Lemniscate, where players battle through waves of three distinct enemy types before facing an intense boss showdown. The game was built using Unity, with all gameplay mechanics and AI behavior scripted in C#. A key feature of the game is the behavior tree system I implemented, which dynamically controls enemy and boss actions, creating challenging and adaptive gameplay experiences.

Key Features:

  • Advanced AI Behavior: Designed and implemented a behavior tree system to control enemy and boss decision-making, allowing for dynamic and strategic combat encounters.
  • Challenging Boss Battle: A climactic encounter that pushes players' skills and strategic thinking.
  • Smooth and Engaging Gameplay: Developed responsive controls and optimized mechanics to deliver an immersive experience within a compact gameplay structure.

This project provided hands-on experience in game development, AI programming, and optimizing gameplay performance within the Unity engine, reinforcing my skills in C# scripting and game design principles.

Finacial Tracker

As part of a collaborative team, I developed gogoMoney, a web application designed to manage personal finances. The app features income and expense tracking, goal setting, metadata documentation, financial projections, and visualized reports. Built using Spring MVC and Hibernate, the application follows a 3-layer architecture to ensure robust functionality and scalability.

Key Contributions:

  • Backend Development: Focused on backend functionality, including database design, query optimization, and ensuring efficient data retrieval and management.
  • Database Integration: Developed and managed the database schema, ensuring seamless interaction between the application and the underlying data.
  • Financial Data Processing: Implemented features for tracking and analyzing financial data, supporting projections and reports that help users make informed decisions.

This project allowed me to strengthen my skills in backend development, database management, and working with Java-based frameworks, while collaborating effectively in a team-oriented Agile environment.
